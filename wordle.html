Joc Wordle

Obiective

Scopul acestui proiect a fost recrearea celebrului joc Wordle, care de data aceasta are doua optiuni:

  1. Modul normal de joc
  2. Un joc ce afiseaza sugestii de cuvinte, astfel incat ghicirea cuvantului secret sa aiba loc cat mai rapid.

Realizare

Aplicatia este realizata pe baza tehnologiilor web (HTML, CSS, Javascript, Flask). Optimizarea procesului de ghicire a cuvantului secret a fost realizata prin implementarea in Python a unor functii ce au la baza criteriul entropiei. Astfel, se calculeaza probabilitatea de aparitie in lista de cuvinte a fiecarei litere din alfabet, care este apoi folosita pentru a determina entropia fiecarui cuvant din lista. Folosind feedback-ul primit in urma unei incercari de ghicire, lista de cuvinte din care este aleasa urmatoarea incercare este filtrata astfel incat sa mai contina doar cuvinte cat mai apropiate de forma cuvantului secret. Sugestia finala este cuvantul care indeplineste toate conditiile de mai sus si care are entropie maxima. In final, numarul mediu de incercari pentru ghicirea tuturor cuvintelor este 4.58.
